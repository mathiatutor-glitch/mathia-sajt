◆ Free sample

Try three problems now

Click "Show solution" to see the method — this is what working with the tutor feels like.

1 · Solve: x² − 5x + 6 = 0
Discriminant: D = (−5)² − 4·6 = 1. Then x = (5 ± 1)/2. Solutions: x = 3 and x = 2. (Vieta check: sum 5, product 6 ✓)
2 · Solve: √(x + 7) = x + 1
Condition: x ≥ −1. Square: x + 7 = x² + 2x + 1 → x² + x − 6 = 0 → x = 2 or x = −3. Reject −3. Solution: x = 2. (√9 = 3 = 2 + 1 ✓)
3 · Solve: log₂ x + log₂(x − 2) = 3
Domain: x > 2. log₂(x(x − 2)) = 3 → x² − 2x = 8 → x² − 2x − 8 = 0 → x = 4 or x = −2. Solution: x = 4.
