1 HTML — kostur stranice

HTML opisuje sadržaj stranice pomoću tagova u uglastim zagradama. Većina tagova ide u paru: otvoreni <p> i zatvoreni </p>.

<!DOCTYPE html> <html> <head> <title>Moja strana</title> </head> <body> <h1>Zdravo!</h1> </body> </html>

head su podaci o stranici (naslov u tabu), a body je ono što se vidi.

Ada kaže

Sačuvaj fajl kao index.html i otvori ga duplim klikom — otvoriće se u pregledaču. 🌟

Probaj sam: dodaj još jedan naslov manje veličine.

Ispod <h1> dodaj <h2>Podnaslov</h2> — h1 je najveći, h6 najmanji.

2 Tekst, linkovi i slike

Najčešći tagovi za sadržaj:

<h1>Naslov</h1> <p>Pasus teksta.</p> <a href="https://mathia.rs">link</a> <img src="slika.jpg" alt="opis slike"> <ul> <li>stavka liste</li> </ul>

href je adresa linka, src je putanja do slike, a alt je opis slike (važan kad se slika ne učita).

Ada kaže

Tagovi <img> i <br> nemaju zatvoreni par — sami se zatvaraju. 💡

Probaj sam: napravi link ka svom omiljenom sajtu.

<a href="https://...">Moj sajt</a> — tekst između tagova je ono što se vidi.

3 CSS — boje i fontovi

CSS daje izgled. Pišemo ga unutar <style> u head. Biramo element, pa mu zadamo osobine.

<style> body { background: #fff8ef; color: #1f3357; font-family: sans-serif; } h1 { color: #c59a3a; } </style>

Svaka osobina je ime: vrednost;. Ne zaboravi ; na kraju.

Rešen primer
p { font-size: 18px; line-height: 1.6; }

Ovo sve pasuse (<p>) pravi većim i prozračnijim. ✔️

Ada kaže

Boje možeš pisati imenom (gold), heksadecimalno (#c59a3a) ili rgb(197,154,58). 🌟

Probaj sam: oboj pozadinu u svetlo plavu.

body { background: lightblue; }

4 CSS — raspored (flexbox)

Da bi elementi stajali jedan pored drugog i bili lepo raspoređeni, koristimo flexbox.

<div class="red"> <div>A</div> <div>B</div> <div>C</div> </div> <style> .red { display: flex; gap: 12px; justify-content: center; } </style>

display: flex reda decu u red; gap je razmak; justify-content: center ih centrira.

Ada kaže

Klasa se u HTML‑u zadaje sa class="red", a u CSS‑u se bira sa tačkom: .red. 💡

Probaj sam: razmakni elemente na krajeve reda.

justify-content: space-between;

5 JavaScript — osnove

JavaScript daje ponašanje — stranica može da računa i reaguje. Pišemo ga u <script>.

<script> let ime = "Ana"; let broj = 5; console.log(ime, broj); function pozdrav(ko) { return "Zdravo, " + ko; } console.log(pozdrav(ime)); </script>

let pravi promenljivu, function funkciju, a console.log ispisuje u konzolu pregledača (F12).

Ada kaže

Konzolu otvoriš tasterom F12 → kartica „Console". Tu vidiš ispis i greške. 🌟

Probaj sam: napravi funkciju koja sabira dva broja.

function zbir(a, b){ return a + b; } pa console.log(zbir(3, 4));

6 Događaji i DOM

DOM je stranica viđena iz JavaScripta. Možemo da uhvatimo element i promenimo ga — najčešće na klik.

<button onclick="promeni()">Klikni me</button> <p id="poruka">Zdravo</p> <script> function promeni() { document.getElementById("poruka") .textContent = "Promenjeno! 🎉"; } </script>

getElementById nađe element po id, a textContent menja njegov tekst.

Rešen primer
<button onclick="alert('Zdravo!')">Pozdrav</button>

alert otvori mali prozor sa porukom. Najbrži način da vidiš da klik radi. ✔️

Ada kaže

Da bi JavaScript „video" dugme, element mora imati id, a funkcija isto ime kao u onclick. 💡

Probaj sam: napravi dugme koje menja boju pasusa.

U funkciji: document.getElementById("poruka").style.color = "red";
