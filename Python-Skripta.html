Python — od prve linije koda

Osnove Pythona, lekcija po lekcija — sa primerima koje možeš odmah da otkucaš i probaš. Ada je uz tebe; kad zapneš, otvori čet i pitaj. 💛

Šta sve učimo

  1. Promenljive i tipovi
  2. Ispis i unos (print, input)
  3. Uslovi (if / elif / else)
  4. Petlje (for, while)
  5. Liste i rečnici
  6. Funkcije
1 Promenljive i tipovi

Promenljiva je kao kutija sa imenom u koju čuvamo neku vrednost. Vrednost upisujemo znakom =.

ime = "Ana" godine = 12 print(ime, "ima", godine, "godina")

Najčešći tipovi podataka

• str — tekst (u navodnicima): "zdravo"
• int — ceo broj: 12
• float — decimalni broj: 3.14
• bool — tačno/netačno: True / False

Rešen primer

Šta ispisuje ovaj kod?

a = 5 b = 3 print(a + b)

Sabira 5 i 3 i ispisuje 8. ✔️

Ada kaže

Imenu promenljive daj jasno značenje: broj_bodova je bolje od x — kasnije ćeš lakše čitati svoj kod. 🌟

Probaj sam: napravi promenljivu grad sa vrednošću "Novi Sad" i ispiši je.

grad = "Novi Sad" pa print(grad) → ispisuje Novi Sad.

2 Ispis i unos

print() ispisuje na ekran, a input() čeka da korisnik nešto otkuca i to vraća kao tekst.

ime = input("Kako se zoveš? ") print("Zdravo,", ime)

Pažnja: input uvek daje tekst

Ako želiš da računaš sa unetim brojem, pretvori ga u broj sa int() ili float().

Rešen primer
godine = input("Koliko imaš godina? ") godine = int(godine) print("Dogodine imaš", godine + 1)

Bez int() bi godine + 1 prijavilo grešku, jer se tekst i broj ne sabiraju. ✔️

Ada kaže

Kad ti kod „pukne" na input, prvo proveri da li si zaboravio da pretvoriš tekst u broj. To je najčešća greška početnika. 💡

Probaj sam: pitaj korisnika dva broja i ispiši njihov zbir.

a = int(input("Prvi: ")), b = int(input("Drugi: ")), pa print(a + b).

3 Uslovi — if / elif / else

Uslovom kažemo programu da nešto uradi samo ako je nešto tačno. Obrati pažnju na dve tačke : i na uvlačenje (razmaci ispod).

ocena = 4 if ocena == 5: print("Odlican!") elif ocena >= 2: print("Prosao si") else: print("Jos malo vezbe")

Najčešća poređenja: == (jednako), != (različito), <, >, <=, >=.

Rešen primer
broj = 7 if broj % 2 == 0: print("paran") else: print("neparan")

% daje ostatak pri deljenju. 7 % 2 = 1, nije 0 → ispisuje neparan. ✔️

Ada kaže

Jedno = dodeljuje vrednost, a dva == porede. Lako se zameni — pazi! 🌟

Probaj sam: ispiši "veliki" ako je broj veći od 100, inače "mali".

if broj > 100: → print("veliki"), else: → print("mali").

4 Petlje — for i while

Petlja ponavlja kod. for koristimo kad znamo koliko puta, a while dok je nešto tačno.

# ispisuje 0, 1, 2, 3, 4 for i in range(5): print(i)

range(5) daje brojeve od 0 do 4 (pet brojeva, počinje od nule).

broj = 1 while broj <= 3: print(broj) broj = broj + 1
Rešen primer
# zbir brojeva od 1 do 5 zbir = 0 for i in range(1, 6): zbir = zbir + i print(zbir)

range(1, 6) daje 1,2,3,4,5. Zbir = 15. ✔️

Ada kaže

Ako se while nikad ne zaustavlja, verovatno si zaboravio da menjaš promenljivu unutar petlje. 😊

Probaj sam: ispiši sve parne brojeve od 0 do 10.

for i in range(0, 11, 2): → print(i). Treći broj u range je korak (2).

5 Liste i rečnici

Lista čuva više vrednosti po redu. Prvom elementu pristupamo sa [0].

ocene = [5, 4, 5, 3] print(ocene[0]) # prva ocena: 5 ocene.append(5) # dodaj na kraj print(len(ocene)) # koliko ih ima: 5

Rečnik čuva parove „ključ → vrednost":

ucenik = {"ime": "Ana", "razred": 6} print(ucenik["ime"]) # Ana
Rešen primer
# prosek ocena ocene = [5, 4, 5, 2] print(sum(ocene) / len(ocene))

sum sabere sve, len prebroji → 16 / 4 = 4.0. ✔️

Ada kaže

Brojanje u Pythonu počinje od 0, pa je prvi element [0], a ne [1]. 🌟

Probaj sam: napravi listu tri omiljene boje i ispiši drugu po redu.

boje = ["plava", "zlatna", "krem"] pa print(boje[1]) → zlatna.

6 Funkcije

Funkcija je deo koda kome damo ime, pa ga pozivamo kad god treba. Pravimo je rečju def.

def pozdrav(ime): print("Zdravo,", ime) pozdrav("Mika") pozdrav("Ana")

Funkcija može i da vrati rezultat

def zbir(a, b): return a + b rezultat = zbir(3, 4) print(rezultat) # 7
Rešen primer
def povrsina_pravougaonika(a, b): return a * b print(povrsina_pravougaonika(5, 3))

Funkcija vraća 5 · 3 = 15. ✔️ Tako jednu računicu pišeš jednom, a koristiš mnogo puta.

Ada kaže

return vrati vrednost iz funkcije; print samo ispiše. To su dve različite stvari — često se mešaju. 💡

Probaj sam: napiši funkciju koja vraća kvadrat broja.

def kvadrat(n): → return n * n. Poziv: print(kvadrat(6)) → 36.
