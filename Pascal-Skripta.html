Jezik na kom se uči logika programiranja u školi i na takmičenjima. Ada je uz tebe; kad zapneš, otvori čet i pitaj. 💛

1 Struktura programa i ispis

Pascal program ima ime, pa telo između begin i end. (sa tačkom!). Ispisujemo sa writeln.

program Pozdrav; begin writeln('Zdravo!'); end.

writeln ispiše i pređe u novi red; write ostaje u istom redu. Tekst ide pod jednostruke navodnike.

Ada kaže

Krajnji end. ima tačku, a unutrašnji blokovi end; imaju tačku‑zarez. To je česta greška! 🌟

Probaj sam: ispiši svoje ime u drugom redu.

Dodaj writeln('Ana'); ispod prvog reda.

2 Promenljive i tipovi

Promenljive se najavljuju u var bloku pre begin. Vrednost se dodeljuje sa :=.

var godine: integer; prosek: real; slovo: char; ime: string; begin godine := 12; prosek := 4.75; end.
Rešen primer
var a, b: integer; begin a := 5; b := 3; writeln(a + b); // 8 end.

Više promenljivih istog tipa može u jednom redu. ✔️

Ada kaže

:= je dodela („postani"), a obično = je poređenje. Lako se pobrkaju! 💡

Probaj sam: napravi promenljivu „zbir" i upiši 10.

U var: zbir: integer; pa u telu zbir := 10;

3 Unos i operatori

Za unos sa tastature koristimo readln.

var a, b: integer; begin readln(a, b); writeln('Zbir: ', a + b); end.

Operatori: + - *, a za cele brojeve div (celobrojno deljenje) i mod (ostatak): 7 div 2 = 3, 7 mod 2 = 1. Za realno deljenje koristi se /.

Ada kaže

U writeln možeš spojiti tekst i broj zarezom: writeln('Rezultat: ', x); 🌟

Probaj sam: učitaj broj i ispiši njegov ostatak pri deljenju sa 3.

readln(n); writeln(n mod 3);

4 Grananje (if, case)

Odluku pravimo sa if … then … else. Pažnja: ispred else nema tačke‑zareza.

if ocena >= 5 then writeln('odlican') else writeln('jos vezbe');

Kad ima više slučajeva, lepši je case:

case dan of 1: writeln('ponedeljak'); 2: writeln('utorak'); else writeln('drugi dan'); end;
Ada kaže

Poređenja: =, <> (različito), <, >, <=, >=. Za „i/ili": and / or. 💡

Probaj sam: ispiši „paran" ako je broj deljiv sa 2.

if n mod 2 = 0 then writeln('paran') else writeln('neparan');

5 Petlje (for, while, repeat)

Tri vrste petlji:

// poznat broj koraka for i := 1 to 5 do writeln(i); // dok važi uslov while i < 5 do begin writeln(i); i := i + 1; end; // bar jednom, pa proveri repeat readln(x); until x = 0;
Rešen primer
var i, zbir: integer; begin zbir := 0; for i := 1 to 5 do zbir := zbir + i; writeln(zbir); // 15 end.

Kad u petlji ima više naredbi, obavij ih u begin … end;. ✔️

Ada kaže

for kad znaš koliko puta; while kad zavisi od uslova; repeat kad treba bar jednom da se izvrši. 🌟

Probaj sam: ispiši brojeve od 10 do 1 unazad.

for i := 10 downto 1 do writeln(i);

6 Nizovi i potprogrami

Niz čuva više vrednosti; indeksi se zadaju opsegom:

var a: array[1..3] of integer; begin a[1] := 5; a[2] := 4; writeln(a[1]); // 5 end.

Potprogrami: function vraća vrednost, procedure ne vraća.

function zbir(a, b: integer): integer; begin zbir := a + b; end;
Ada kaže

U funkciji rezultat „vratiš" tako što imenu funkcije dodeliš vrednost: zbir := a + b; 💡

Probaj sam: napiši funkciju koja vraća kvadrat broja.

function kvadrat(n: integer): integer; begin kvadrat := n * n; end;
