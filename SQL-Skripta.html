SQL — razgovor s bazom

Kako da iz tabela izvučeš tačno ono što ti treba. Ada je uz tebe; kad zapneš, otvori čet i pitaj. 💛

Šta sve učimo

  1. Tabele i SELECT
  2. WHERE — filtriranje
  3. ORDER BY — sortiranje
  4. Brojanje i prosek (agregati)
  5. JOIN — spajanje tabela
  6. Unos, izmena, brisanje

U primerima koristimo tabelu ucenici:

idimerazredprosek
1Ana54.80
2Marko53.90

1 Tabele i SELECT

Baza čuva podatke u tabelama (vrste = redovi, kolone = osobine). Podatke čitamo naredbom SELECT.

SELECT ime, prosek FROM ucenici;

Za sve kolone koristi se zvezdica:

SELECT * FROM ucenici;
Ada kaže

Svaka SQL naredba se završava sa ;. Velika slova za ključne reči (SELECT, FROM) nisu obavezna, ali se lepše čita. 🌟

Probaj sam: izlistaj samo imena svih učenika.

SELECT ime FROM ucenici;

2 WHERE — filtriranje

WHERE bira samo redove koji zadovoljavaju uslov.

SELECT ime FROM ucenici WHERE prosek >= 4.5;

Operatori: =, <> (različito), <, >, <=, >=. Više uslova spaja se sa AND i OR.

Rešen primer
SELECT ime FROM ucenici WHERE razred = 5 AND prosek > 4;

Vraća petake sa prosekom preko 4 → samo „Ana". ✔️

Ada kaže

Tekst se u uslovu piše pod jednostrukim navodnicima: WHERE ime = 'Ana'. 💡

Probaj sam: nađi učenike iz 6. razreda.

SELECT * FROM ucenici WHERE razred = 6;

↑ na vrh
3 ORDER BY — sortiranje

ORDER BY ređa rezultat. ASC = rastuće (podrazumevano), DESC = opadajuće.

SELECT ime, prosek FROM ucenici ORDER BY prosek DESC;

Sada je prvi onaj sa najvećim prosekom.

Ada kaže

Možeš da spojiš: WHERE pa ORDER BY — prvo filtriraš, pa sortiraš. 🌟

Probaj sam: izlistaj učenike po imenu, abecedno.

SELECT ime FROM ucenici ORDER BY ime ASC;

↑ na vrh
4 Brojanje i prosek (agregati)

Agregatne funkcije računaju nad više redova: COUNT, SUM, AVG, MIN, MAX.

SELECT COUNT(*) FROM ucenici; // koliko ih ima SELECT AVG(prosek) FROM ucenici; // prosečan prosek

Uz GROUP BY računaš po grupama:

SELECT razred, COUNT(*) FROM ucenici GROUP BY razred;
Rešen primer

Gornji upit vraća: razred 5 → 2 učenika, razred 6 → 1 učenik. ✔️

Ada kaže

COUNT(*) broji redove; AVG(kolona) računa prosek baš te kolone. 💡

Probaj sam: nađi najveći prosek u tabeli.

SELECT MAX(prosek) FROM ucenici;

5 JOIN — spajanje tabela

Kad su podaci u dve tabele, JOIN ih spaja preko zajedničke kolone. Npr. tabela ocene ima ucenik_id.

SELECT u.ime, o.predmet, o.ocena FROM ucenici u JOIN ocene o ON o.ucenik_id = u.id;

u i o su kratka imena (aliasi) za tabele, da upit bude čitljiviji.

Ada kaže

Ključ je ON — tu kažeš po čemu se tabele povezuju (najčešće id iz jedne = id u drugoj). 🌟

Probaj sam: dodaj uslov da prikažeš samo ocene 5.

Na kraj dodaj WHERE o.ocena = 5;

↑ na vrh
6 Unos, izmena, brisanje

Podatke menjamo sa tri naredbe:

INSERT INTO ucenici (ime, razred, prosek) VALUES ('Petar', 6, 4.20); UPDATE ucenici SET prosek = 5.00 WHERE id = 1; DELETE FROM ucenici WHERE id = 2;
Ada kaže

Pažnja! Bez WHERE, UPDATE i DELETE menjaju/brišu celu tabelu. Uvek prvo proveri WHERE. 🌟

Probaj sam: promeni Markov razred na 6.

UPDATE ucenici SET razred = 6 WHERE ime = 'Marko';
