Java — od osnova do objekata

Osnove Jave lekcija po lekcija, sa primerima koje možeš da prevedeš i pokreneš. Ada je uz tebe; kad zapneš, otvori čet i pitaj. 💛

1 Prvi program i ispis

U Javi sve živi unutar klase. Program kreće od metode main, a ispisujemo sa System.out.println.

public class Main { public static void main(String[] args) { System.out.println("Zdravo!"); } }

println ispiše i pređe u novi red; print ostaje u istom redu.

Ada kaže

Ime fajla mora da se poklapa sa imenom javne klase: klasa Main → fajl Main.java. 🌟

Probaj sam: ispiši još jedan red sa svojim imenom.

Dodaj System.out.println("Ana"); ispod prvog reda.

2 Promenljive i tipovi

Promenljivu prvo deklarišemo (tip), pa dodelimo vrednost.

int godine = 12; double cena = 9.99; char slovo = 'A'; String ime = "Ana"; boolean tacno = true;
Rešen primer
int a = 5, b = 3; System.out.println(a + b); // 8

Tekst (String) ide pod dvostruke navodnike, znak (char) pod jednostruke. ✔️

Ada kaže

String se u Javi piše velikim S — to je klasa, ne osnovni tip. 💡

Probaj sam: napravi double „prosek" i ispiši ga.

double prosek = 4.5; pa System.out.println(prosek);

3 Unos (Scanner) i operatori

Za unos koristimo Scanner, koji prvo uvezemo.

import java.util.Scanner; Scanner sc = new Scanner(System.in); int a = sc.nextInt(); int b = sc.nextInt(); System.out.println(a + b);

Operatori: + - * / %. Za cele brojeve 7 / 2 = 3 (ceo deo), a 7 % 2 = 1 (ostatak).

Ada kaže

nextInt() čita ceo broj, nextDouble() decimalni, nextLine() ceo red teksta. 🌟

Probaj sam: učitaj broj i ispiši njegov kvadrat.

int n = sc.nextInt(); pa System.out.println(n * n);

4 Grananje i petlje

Odluke: if / else. Ponavljanje: for i while.

if (ocena >= 5) { System.out.println("odlican"); } else { System.out.println("jos vezbe"); }
for (int i = 0; i < 5; i++) { System.out.println(i); // 0..4 }
Rešen primer
int zbir = 0; for (int i = 1; i <= 5; i++) { zbir += i; } System.out.println(zbir); // 15

zbir += i je kraće za zbir = zbir + i. ✔️

Ada kaže

Poređenja: ==, !=, <, >, <=, >=. Za „i/ili": && / ||. 💡

Probaj sam: ispiši parne brojeve od 0 do 10.

for (int i = 0; i <= 10; i += 2) System.out.println(i);

5 Nizovi i ArrayList

Niz čuva fiksan broj vrednosti istog tipa:

int[] a = {5, 4, 3}; System.out.println(a[0]); // 5 System.out.println(a.length); // 3

Kad ti treba lista koja raste, koristiš ArrayList:

import java.util.ArrayList; ArrayList<Integer> lista = new ArrayList<>(); lista.add(5); lista.add(4); System.out.println(lista.size()); // 2
Ada kaže

Niz ima .length (bez zagrada), a ArrayList .size() (sa zagradama). Lako se zameni! 🌟

Probaj sam: prođi kroz niz i ispiši sve elemente.

for (int i = 0; i < a.length; i++) System.out.println(a[i]);

6 Metode

Metoda je imenovani deo koda. Navedemo tip rezultata, ime i parametre.

static int zbir(int a, int b) { return a + b; } public static void main(String[] args) { System.out.println(zbir(3, 4)); // 7 }

Ako metoda ne vraća ništa, tip je void.

Rešen primer
static int kvadrat(int n) { return n * n; } // kvadrat(6) vraća 36

Tako jednu računicu pišeš jednom, a koristiš mnogo puta. ✔️

Ada kaže

Metode koje zoveš iz main (koji je static) i same treba da budu static — dok ne pređeš na objekte. 💡

Probaj sam: napiši metodu koja vraća veći od dva broja.

static int veci(int a, int b){ return (a > b) ? a : b; }
