C i C++ — od osnova

Osnove C/C++ lekcija po lekcija, sa primerima koje možeš da prevedeš i pokreneš. Marina je uz tebe; kad zapneš, otvori čet i pitaj. 💛

1 Prvi program i ispis

Svaki C++ program počinje od funkcije main. Za ispis koristimo cout.

#include <iostream> using namespace std; int main() { cout << "Zdravo!" << endl; return 0; }

<< šalje tekst u ispis, a endl prelazi u novi red.

Marina kaže

U C++‑u skoro svaka naredba se završava sa ;. Kad ti prijavi grešku „expected ;", proveri prethodnu liniju. 🌟

Probaj sam: ispiši svoje ime u novom redu posle pozdrava.

Dodaj još jedan red: cout << "Ana" << endl; ispod prvog.

2 Promenljive i tipovi

Promenljivu prvo deklarišemo (kažemo tip), pa joj dodelimo vrednost.

int godine = 12; // ceo broj double cena = 9.99; // decimalni char slovo = 'A'; // jedan znak string ime = "Ana"; // tekst bool tacno = true; // tačno/netačno
Rešen primer
int a = 5, b = 3; cout << a + b; // 8

Dve promenljive istog tipa možemo deklarisati u jednoj liniji. ✔️

Marina kaže

Za tekst (string) dodaj #include <string> na vrh. 💡

Probaj sam: napravi double promenljivu „prosek" i ispiši je.

double prosek = 4.5; pa cout << prosek;

3 Unos i operatori

Za unos sa tastature koristimo cin i >>.

int a, b; cout << "Unesi dva broja: "; cin >> a >> b; cout << "Zbir je " << a + b;

Računski operatori: + - * / % (ostatak). Pažnja: 7 / 2 = 3 za cele brojeve (ceo deo!), a 7 % 2 = 1.

Marina kaže

Ako ti treba pravo deljenje, bar jedan broj neka bude double: 7.0 / 2 = 3.5. 🌟

Probaj sam: učitaj broj i ispiši njegov kvadrat.

int n; cin >> n; cout << n * n;

4 Grananje i petlje

Odluke pravimo sa if / else, ponavljanje sa for i while.

if (ocena >= 5) { cout << "odlican"; } else { cout << "jos vezbe"; }
for (int i = 0; i < 5; i++) { cout << i << " "; // 0 1 2 3 4 }
Rešen primer
// zbir brojeva od 1 do 5 int zbir = 0; for (int i = 1; i <= 5; i++) zbir += i; cout << zbir; // 15

zbir += i je kraće za zbir = zbir + i. ✔️

Marina kaže

U for petlji: početak i=0, uslov i<5, korak i++ — tri dela, razdvojena sa ;. 💡

Probaj sam: ispiši parne brojeve od 0 do 10.

for (int i = 0; i <= 10; i += 2) cout << i << " ";

5 Nizovi i vektori

Niz čuva više vrednosti istog tipa. Prvom elementu pristupamo sa [0].

int a[3] = {5, 4, 3}; cout << a[0]; // 5

U C++‑u je praktičniji vector jer može da raste:

#include <vector> vector<int> v = {5, 4, 3}; v.push_back(2); // dodaj na kraj cout << v.size(); // 4
Marina kaže

Brojanje počinje od 0, pa niz sa 3 elementa ima indekse [0], [1], [2]. 🌟

Probaj sam: prođi kroz vektor i ispiši sve elemente.

for (int i = 0; i < v.size(); i++) cout << v[i] << " ";

6 Funkcije

Funkcija je imenovani deo koda. Navedemo tip rezultata, ime i parametre.

int zbir(int a, int b) { return a + b; } int main() { cout << zbir(3, 4); // 7 return 0; }

Ako funkcija ne vraća ništa, tip je void.

Rešen primer
int kvadrat(int n) { return n * n; } // kvadrat(6) vraća 36

Jednu računicu napišeš jednom, a koristiš koliko god puta. ✔️

Marina kaže

Funkciju definiši iznad main (ili najavi prototip), da je program „vidi". 💡

Probaj sam: napiši funkciju koja vraća veći od dva broja.

int veci(int a, int b){ return (a > b) ? a : b; } — ili sa if/else.
