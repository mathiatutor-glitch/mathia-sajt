Kotlin — moderan jezik za Android i više

Kotlin je sažet, bezbedan i prijatan za pisanje — popularan za Android. Lekcija po lekcija, sa primerima koje odmah pokreneš. Ada je uz tebe. 💛

1 val/var i tipovi

val je nepromenljiva vrednost (kao konstanta), var se može menjati. Tip se obično izvodi sam.

val ime = "Ana" ne menja se var broj = 5 može da se menja val pi: Double = 3.14 eksplicitan tip broj = 10 ok, jer je var
Rešen primer

val ime = "Ana" ne može da se promeni; ime = "Mia" daje grešku. Treba li menjati — koristi var. ✔️

Ada kaže

Počni sa val; pređi na var samo kad stvarno menjaš vrednost. Manje grešaka. 🌟

Napravi promenljivu grad koja može da se menja.

var grad = "Niš".

2 Null-bezbednost

Kotlin razlikuje tip koji sme biti null (String?) od onog koji ne sme (String) — tako sprečava pad programa.

var a: String = "tekst" ne sme null var b: String? = null sme null val n = b?.length bezbedan poziv → null val d = b ?: "prazno" podrazumevana vrednost
Rešen primer

b?.length vraća null ako je b null (ne pukne). b ?: "prazno" daje „prazno" kad je b null. ✔️

Ada kaže

Znak ? je prijatelj — pita „ako nije null". Izbegavaj !! jer forsira i ruši program. 💡

Kako bezbedno uzmeš dužinu od ime: String?

ime?.length.

3 Funkcije

Funkciju praviš rečju fun. Kratke mogu kao izraz sa =, bez { } i return.

fun saberi(a: Int, b: Int): Int { return a + b } fun kvadrat(x: Int) = x * x kratka saberi(3, 4) 7
Rešen primer

kvadrat(5) = 25. Kratka forma sa = ne traži ni return ni zagrade. ✔️

Ada kaže

Imenovani argumenti čine poziv jasnim: saberi(a = 3, b = 4) — korisno kad ima više parametara. 🌟

Napiši kratku funkciju koja vraća dvostruko.

fun dupliraj(x: Int) = x * 2.

4 when i petlje

when je moćniji switch. for ide kroz opseg ili kolekciju.

when (ocena) { 5 -> println("odlican") 4 -> println("vrlo dobar") else -> println("prolaz") } for (i in 1..5) println(i) 1 do 5
Rešen primer

when proverava vrednost i bira granu; else hvata ostalo. 1..5 je opseg uključivo. ✔️

Ada kaže

when može i da vrati vrednost: val opis = when {...} — tada je izraz, ne samo grananje. 💡

Ispiši brojeve od 1 do 3 petljom.

for (i in 1..3) println(i).

5 Kolekcije (map, filter)

listOf pravi nepromenljivu listu, mutableListOf promenljivu. map i filter rade kao drugde.

val brojevi = listOf(1, 2, 3, 4) brojevi.map { it * 2 } [2,4,6,8] brojevi.filter { it % 2 == 0 } [2,4] brojevi.sum() 10
Rešen primer

it je podrazumevano ime elementa u lambdi. brojevi.filter { it > 2 } → [3, 4]. ✔️

Ada kaže

it ti štedi pisanje: { it * 2 } umesto { x -> x * 2 }. Koristi kad je jedan parametar. 🌟

Od [1,2,3] napravi listu kvadrata.

brojevi.map { it * it }.

6 Klase i data klase

class je za objekte; data class za nosioce podataka (automatski lep ispis i poređenje).

class Pas(val ime: String) { fun laji() = "$ime laje" } data class Tacka(val x: Int, val y: Int) val p = Pas("Reks") val t = Tacka(2, 3)
Rešen primer

data class Tacka automatski ima lep ispis: println(t) → Tacka(x=2, y=3). Obična klasa to nema. ✔️

Ada kaže

"$ime" ubacuje vrednost u tekst (string-templejt). Za izraze: "${a + b}". 💡

Napravi data klasu Knjiga sa naslovom i godinom.

data class Knjiga(val naslov: String, val godina: Int).
