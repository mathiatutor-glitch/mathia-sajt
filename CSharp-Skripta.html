C# — od konzole do objekata

Osnove C# (čita se „si šarp") lekcija po lekcija. Marina je uz tebe; kad zapneš, otvori čet i pitaj. 💛

1 Prvi program i ispis

Program kreće od metode Main unutar klase. Ispisujemo sa Console.WriteLine.

using System; class Program { static void Main() { Console.WriteLine("Zdravo!"); } }

WriteLine ispiše i pređe u novi red; Write ostaje u istom redu.

Marina kaže

Skoro svaka naredba se završava sa ;. C# razlikuje velika i mala slova: Console mora baš tako. 🌟

Probaj sam: ispiši još jedan red sa svojim imenom.

Dodaj Console.WriteLine("Ana"); ispod prvog reda.

2 Promenljive i tipovi

Promenljivu deklarišemo (tip), pa joj dodelimo vrednost.

int godine = 12; double cena = 9.99; char slovo = 'A'; string ime = "Ana"; bool tacno = true;
Rešen primer
int a = 5, b = 3; Console.WriteLine(a + b); // 8

Tekst (string) pod dvostrukim, znak (char) pod jednostrukim navodnicima. ✔️

Marina kaže

Ako ne želiš da pišeš tip, koristi var: var ime = "Ana"; — C# sam pogodi tip. 💡

Probaj sam: napravi double „prosek" i ispiši ga.

double prosek = 4.5; pa Console.WriteLine(prosek);

3 Unos i operatori

Console.ReadLine uvek vraća tekst, pa ga pretvorimo u broj.

Console.Write("Unesi broj: "); int n = int.Parse(Console.ReadLine()); Console.WriteLine(n * 2);

Operatori: + - * / %. Za cele brojeve 7 / 2 = 3 (ceo deo), 7 % 2 = 1 (ostatak).

Marina kaže

Za decimalni unos koristi double.Parse(...). Za pravo deljenje neka bar jedan broj bude double. 🌟

Probaj sam: učitaj broj i ispiši njegov kvadrat.

int n = int.Parse(Console.ReadLine()); Console.WriteLine(n * n);

4 Grananje i petlje

Odluke: if / else. Ponavljanje: for i while.

if (ocena >= 5) { Console.WriteLine("odlican"); } else { Console.WriteLine("jos vezbe"); }
for (int i = 0; i < 5; i++) { Console.WriteLine(i); // 0..4 }
Rešen primer
int zbir = 0; for (int i = 1; i <= 5; i++) { zbir += i; } Console.WriteLine(zbir); // 15

zbir += i je kraće za zbir = zbir + i. ✔️

Marina kaže

Poređenja: ==, !=, <, >, <=, >=. Za „i/ili": && / ||. 💡

Probaj sam: ispiši parne brojeve od 0 do 10.

for (int i = 0; i <= 10; i += 2) Console.WriteLine(i);

5 Nizovi i liste

Niz čuva fiksan broj vrednosti:

int[] a = {5, 4, 3}; Console.WriteLine(a[0]); // 5 Console.WriteLine(a.Length); // 3

Kad ti treba lista koja raste, koristiš List:

using System.Collections.Generic; List<int> lista = new List<int>(); lista.Add(5); lista.Add(4); Console.WriteLine(lista.Count); // 2
Marina kaže

Niz ima .Length, a lista .Count — i jedno i drugo počinje od indeksa 0. 🌟

Probaj sam: prođi kroz niz i ispiši sve elemente.

for (int i = 0; i < a.Length; i++) Console.WriteLine(a[i]);

6 Metode i klase (OOP)

Metoda je imenovani deo koda:

static int Zbir(int a, int b) { return a + b; } // Zbir(3, 4) vraća 7

Klasa je nacrt objekta — ima polja (podatke) i metode (ponašanje):

class Pas { public string ime; public void Laji() { Console.WriteLine(ime + ": Av!"); } } // upotreba: Pas p = new Pas(); p.ime = "Reks"; p.Laji(); // Reks: Av!
Marina kaže

Klasa je kao kalup za kolače, a objekat (new Pas()) je jedan konkretan kolač. C# se mnogo koristi i za igre u Unity — sve kreće baš od klasa. 🌟

Probaj sam: dodaj klasi Pas polje „godine" i ispiši ga u Laji().

Dodaj public int godine; i u Laji() Console.WriteLine(godine);
