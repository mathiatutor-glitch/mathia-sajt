1 Šema u OrCAD Capture

U Capture-u praviš projekat i crtaš šemu. Novi projekat: File → New → Project (PSpice Analog or Mixed).

Postavi deo → Place → Part P Žica → Place → Wire W Masa → simbol „0"
Rešen primer

Za RC kolo postavi R, C, izvor (VDC ili VSIN) i masu 0. ✔️

Višnja kaže

Masa u PSpice mora biti 0 (net 0). Bez nje simulacija ne radi. 🌟

Kako postavljaš deo u Capture-u?

Place → Part (prečica P).

2 Simboli i biblioteke

Delovi su u bibliotekama (.olb). Za simulaciju koristi PSpice biblioteke (analog, source). Dodaješ ih u Place Part dijalogu.

Place Part → Add Library → analog.olb, source.olb Često: R, C, L, VDC, VSIN, VAC, Q2N2222 Masa: deo „0"
Rešen primer

Sinusni izvor → VSIN. Stalni napon → VDC. ✔️

Višnja kaže

Za PSpice biraj delove iz PSpice biblioteka — obični Capture delovi nemaju model za simulaciju. 💡

Koji izvor koristiš za sinus?

VSIN.

↑ na vrh
3 PSpice simulacije

Napraviš Simulation Profile (PSpice → New Simulation Profile) i izabereš tip analize.

Transient : Run to time = 5ms vreme AC Sweep : 1 Hz … 1 MHz frekvencije DC Sweep : menjaj napon 0…10 V Monte Carlo: rasipanje vrednosti Pokreni: PSpice → Run
Rešen primer

Odziv RC kola u vremenu → Transient, Run to 5 ms. ✔️

Višnja kaže

Stavi Voltage Marker na čvor pre pokretanja — odmah vidiš talas u Probe-u. 🌟

Koja analiza za filtar (frekvencije)?

AC Sweep.

4 Analiza rezultata

Posle Run otvara se Probe (grafik). Signal dodaješ sa Trace → Add Trace ili markerima u šemi.

Probe → Trace → Add Trace Markeri u šemi → napon, struja, dB Kursori → Trace → Cursor izmeri tačku
Rešen primer

Pojačanje filtra → DB(V(out)/V(in)) u AC analizi (Bodeov dijagram). ✔️

Višnja kaže

Prazan grafik? Proveri da li je analiza prošla bez greške (Output prozor) i da li ima markera. 💡

Kako vidiš pojačanje u decibelima?

DB(V(out)/V(in)).

↑ na vrh
5 Prelazak na ploču (Allegro)

Posle šeme i simulacije praviš netlist i prelaziš u PCB Editor (Allegro/OrCAD PCB). Svakom delu treba PCB Footprint.

Svakom delu → svojstvo PCB Footprint Tools → Create Netlist PCB Editor → učitaj netlist → raspored → rutiranje
Rešen primer

Bez dodeljenog footprinta deo se neće preneti na ploču — dodeli ga svakom. ✔️

Višnja kaže

Profesionalni tok razdvaja simulaciju (PSpice) i layout (Allegro) — ista šema služi za oba. 🌟

Šta praviš pre layout-a?

Netlist (i dodeljene footprints).

↑ na vrh
6 Pravila i proizvodni izlaz

U Constraint Manager-u postavljaš pravila (širine, razmaci, impedansa). Proveravaš DRC-om, a izrađuješ Gerber + drill.

Constraint Manager → pravila (širine, razmaci) Provera → DRC Izlaz → Manufacture → Artwork (Gerber) + NC Drill
Rešen primer

Za hobi ploču dovoljna su osnovna pravila širine i razmaka; za brze signale postavljaš impedansu. ✔️

Višnja kaže

Cadence je moćan, ali složen — kreni od malog kola, savladaj tok, pa rasti. Tu sam za svaki korak. 💛

Gde podešavaš pravila dizajna?

U Constraint Manager-u.
